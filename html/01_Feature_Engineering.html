# Definimos la funcin de transformacin de la nota FICO
def rating_fico (x, y):
    if  x >= 800 & y <= 850:
        return 'Exceptional'
    elif x >= 749 & y <= 799:
        return 'Very good'
    elif x >= 670 & y <= 739:
        return 'Good'
    elif x >= 580 & y <= 669:
        return 'Fair'
    else:
        return 'Very poor'

Bancarrotas y registros públicos¶
